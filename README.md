# Cisco Packet Tracer Network Labs

This repository contains Cisco Packet Tracer labs and networking exercises that I practiced during my networking and cybersecurity studies.

The labs include basic network configuration, routing, switching, troubleshooting, and simple security-related configurations.

## Topics

- Basic Router and Switch Configuration
- IPv4 Addressing and Subnetting
- VLAN and Trunk Configuration
- Inter-VLAN Routing
- Static Routing
- OSPF
- DHCPv4 and DHCPv6
- NAT Configuration
- ACL (Access Control List)
- STP and EtherChannel
- Wireless LAN Basics
- Network Troubleshooting
- Basic Network Security
